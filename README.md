# Part I - Instructions
-----
# Frontend- React.js & Redux & Redux Thunk:
1. cd client
2. npm install

⭐ To Run:
- npm run start

**The app will run on this url: http://localhost:3000**

# Backend- Node.Js
1. cd server
2. npm install

⭐ To Run:
- node app

⭐ Run Unit test - chai & mocha:
- it's on folder called test.
- npm test

**The server will run on this url: http://localhost:8080**
